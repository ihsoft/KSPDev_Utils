A Lingoona Grammar template in the current language.
Remarks

The template is resolved via the Localizer only once. The resolved value is cached and re-sued.

If there is no string defined for the tag, then a defaultTemplate will be used. A warning record will be logged to help tracking such issues.

When current language is changed the cached version needs to be reloaded. Call the LoadLocalization method to force it. However, as of KSP 1.3.0 the language cannot be changed while the game is running.
